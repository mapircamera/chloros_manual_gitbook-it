# Formati immagine di output

### TIFF (16 bit)

Output in formato TIFF (.tif) a 16 bit (numero digitale pixel 0 - 65535).

### TIFF (32 bit, percentuale%)

Output in formato TIFF (.tif) a 32 bit (pixel float 0,0 - 1,0). I valori dei pixel rappresentano le percentuali di riflettanza dove 0,0 = 0% e 1,0 = 100%.

### **JPG (8 bit)**

Output in formato JPG (.jpg) a 8 bit (numero digitale pixel 0 -255).

### **PNG (8 bit)**

Output in formato PNG (.png) a 8 bit (numero digitale pixel 0 - 255). Compressione senza perdita adatta per la visualizzazione e la visualizzazione web.
