# Formati immagine di output

### TIFF (16 bit)

16 bit (numero digitale pixel 0 - 65535) Output in formato TIFF (.tif).

### TIFF (32 bit, percentuale %)

32 bit (pixel float 0,0 - 1,0) TIFF (.tif) formato di output. I valori dei pixel rappresentano le percentuali di riflettanza dove 0,0 = 0% e 1,0 = 100%.

### **JPG (8 bit)**

Formato di output JPG (.jpg) a 8 bit (numero digitale pixel 0 -255).

### **PNG (8 bit)**

Formato di output PNG (.png) a 8 bit (numero digitale pixel 0 - 255). Compressione senza perdita di dati adatta alla visualizzazione sul web.
